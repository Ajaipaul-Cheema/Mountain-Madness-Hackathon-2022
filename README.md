# Backend Startup Instructions

### Requirements
- Python 3.7+

### Instructions
## *Using BASH*
1. `pip install virtualenv`
2. `python -m virtualenv venv`
3. `source venv/scripts/activate`
4. `pip install -r requirements.txt`
5. `uvicorn main:app --reload`
